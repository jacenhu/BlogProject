# 书籍推荐及我的2021年阅读计划
## 看过的书
列举一下近几年看过觉得比较好的文学类的书：

这里说的好，有故事性、启发性等。

我想，一千个读者就有一千个哈姆雷特。

不要期望一定从阅读中得到什么，影响是潜移默化的，就像的复旦的“自由而无用”。

（1）东野圭吾
* 《白夜行》
* 《解忧杂货铺》
* 《嫌疑人X的献身》

（2）治愈系
* 《岛上书店》
* 《追风筝的人》
* 《群山回响》
* 《灿烂千阳》
* 《偷影子的人》

（3）实用、社科类
* 《原则》
* 《乌合之众》
* 《非暴力沟通》
* 《富爸爸穷爸爸》
* 《高效能人士的七个习惯》
* 《断舍离》

（4）纯文学
* 《简爱》
* 《黄金时代》
* 《人间失格》
* 《月亮与六便士》
* 《小王子》
